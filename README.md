# Introdução à Análise de Dados 📊

Este repositório foi desenvolvido com o objetivo de registrar o aprendizado, práticas e projetos relacionados aos fundamentos de **Análise de Dados** utilizando a linguagem **Python**. Aqui são abordados conceitos essenciais para a manipulação, limpeza, exploração estruturada e visualização de dados.

---

## 🚀 Conteúdos e Recursos Abordados

O projeto engloba as principais etapas que compõem o ciclo de vida de uma análise de dados:

* **Manipulação e Estruturação de Dados:** Uso de estruturas como Series e DataFrames para ler, filtrar, agrupar e transformar dados brutos.
* **Limpeza de Dados (Data Cleaning):** Tratamento de valores ausentes (NaN), remoção de registros duplicados, conversão de tipos de dados e correção de inconsistências estruturais.
* **Análise Exploratória de Dados (EDA):** Identificação de padrões, comportamento de variáveis, cálculo de estatísticas descritivas (média, mediana, desvio padrão) e análise de correlações.
* **Visualização de Dados:** Construção de gráficos informativos e estéticos para extrair insights e comunicar resultados de forma clara.

---

## 🛠️ Tecnologias e Ferramentas

As principais ferramentas e bibliotecas utilizadas neste ecossistema de análise são:

* **Linguagem:** Python 3.x
* **Análise e Manipulação:** [Pandas](https://pandas.pydata.org/) e [NumPy](https://numpy.org/)
* **Visualização de Dados:** [Matplotlib](https://matplotlib.org/) e [Seaborn](https://seaborn.pydata.org/)
* **Ambiente de Desenvolvimento:** Jupyter Notebook
